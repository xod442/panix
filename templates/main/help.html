Help for Panix

Panix help information

Assumptions

You have a Palo Alto Firewall on your network.

You must have credentials to modify this firewall

CSV file you want to upload MUST look like this!

  1. IP address of FW management interface
  2. IP network for the subinterface...not the vlan gateway address...ends in zero!
  3. Column "C" not used....don't ask :-)
  4. Subnet mask bits as shown...full mask with slash followed by bits
  5. Vlan number
  6. Comments about the sub interface
  7. Security Zone

This app was written to bulk add Layer 3 subinterfaces to a Palo Alto firewall using the pandevice python library.

Current Behavior

  • You must have a separate CSV file for each firewall or one giant one that contains all the data
  • A multisheet excel file will not work
  • Upload CSV's into the mongo database
  • Deploy firewall subinterfaces one firewall at a time
  • If you have trouble logging in go to http://localhost:5001/logout
